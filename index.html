About¶

We provide a specification for targetscope entities and predicates, which helps describe terminology used in the JSON schema and in web service requests used by the targetscope front-end.

Predicates are modeled on BEDOPS set and map operations described in the documentation for bedops and bedmap binaries (see the BEDOPS documentation for more detail), while data entities are derived from columnar data found in BED files, or are composed into compound elements by using base types as building blocks.

The TFBS example describes a complete pipeline of predicates and data entities used to find the binding sites of a transcription factor. This example describes one way (in broad terms) how the components of this specification document could be used to construct and perform tasks in the targetscope application.
